I was working on the covid data visualization and had scraped the districts of India. I thought of using the same CSV file to extract the weather information from all districts across India. I have used beautifulsoup to scrape the data from darksky website.
Libraries used: BeautifulSoup(scraping tool), urllib(fetching data across web),pandas(reading csv file), csv(for writing in the files), time(for the page to load)
